# Quality Gates

## Layout & Hierarchy
- One clear primary focal point per section.
- Sections have consistent vertical rhythm and alignment.
- Primary CTA stands out without competing accents.

## Typography & Spacing
- Consistent type scale and line heights.
- Readable line length on desktop and mobile.
- Spacing uses a repeatable scale.

## Color & Contrast
- Text contrast is strong enough for readability.
- Accents are used sparingly and consistently.
- Backgrounds support content, not distract from it.

## Components & States
- Hover, focus, and active states are present for interactive elements.
- Empty and loading states are considered where relevant.
- Components feel cohesive and reusable.

## Accessibility
- Semantic HTML used (headings in order, buttons for actions, links for navigation).
- Focus states are visible and not removed.
- Interactive elements have adequate hit targets.

## Responsiveness
- Mobile layout is deliberate, not a squeezed desktop.
- Key sections reflow cleanly at common breakpoints.

## Performance & Code Quality
- Avoid excessive DOM depth.
- Avoid heavy shadow stacks or expensive effects.
- Class lists are readable and organized.
