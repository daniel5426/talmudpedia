# Concepts Overview

The first public docs wave stays narrow on purpose. You only need three concepts to understand the initial build journey:

- agents decide and orchestrate
- knowledge and retrieval provide grounded context
- tools let the agent act

Read the deeper concept pages only when the quickstart raises a concrete question.
