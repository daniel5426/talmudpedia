# SDK and API Status

The public documentation surface is broader than one SDK.

## Current documented surfaces

- embedded agent runtime
- host-anywhere runtime SDK
- published app runtime behavior

## Current guidance

- use the documented public contract first
- do not rely on internal admin or control-plane shortcuts from external apps
- treat missing SDK capabilities as a public API gap, not as a reason to bypass the contract

## Current maturity

This page is a status marker, not a complete reference. Full generated API reference is intentionally deferred until the public contract set is more stable.
